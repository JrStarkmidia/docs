Você precisa manter alguns cadastros sempre atualizados.

## Cadastros que você usa mais

| Cadastro | Quando usar |
| --- | --- |
| **Clientes** | Antes de criar pedidos para facilitar preenchimento |
| **Produtos** | Quando vende produtos |
| **Tarefas** | Quando vende serviços |
| **Categorias** | Para organizar receitas, despesas e pedidos |
| **Pagamentos** | Para manter as formas de pagamento disponíveis |

## Dica prática

Sempre cadastre primeiro o básico e depois faça os lançamentos.

Exemplo:

1. Cadastre cliente.
2. Cadastre produto ou tarefa.
3. Crie o pedido.
