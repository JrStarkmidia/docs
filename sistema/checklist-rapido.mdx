Use esta pagina como lista de rotina.

<Tip>
Para imprimir: pressione `Ctrl + P` no Windows/Linux ou `Command + P` no Mac.
</Tip>

## Inicio do dia

- [ ] Entrei no sistema com meu usuario.
- [ ] Conferi o **Dashboard diario**.
- [ ] Verifiquei se existem receitas pendentes.
- [ ] Verifiquei se existem despesas pendentes.

## Durante o dia

- [ ] Registrei receitas novas quando necessario.
- [ ] Registrei despesas novas quando necessario.
- [ ] Criei pedidos novos corretamente.
- [ ] Confirmei pagamentos recebidos.
- [ ] Atualizei status de entrega quando necessario.

## Fim do dia

- [ ] Conferi pedidos pendentes.
- [ ] Conferi pedidos confirmados.
- [ ] Revisei os valores principais no dashboard.
- [ ] Tirei duvidas com o suporte, se preciso.

## Erros para evitar

- [ ] Nao deixar pedido sem fechar.
- [ ] Nao esquecer de confirmar pagamento concluido.
- [ ] Nao cadastrar valor sem conferir a data.

## Ajuda rapida

- Se tiver dificuldade, veja [Duvidas frequentes](/sistema/duvidas-frequentes).
- Se precisar de suporte, use `comercial@starkmidia.com`.
